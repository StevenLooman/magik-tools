The nesting level of nodes should not exceed a defined threshold. Complex code may perform poorly and can be difficult to test thoroughly.

Noncompliant Code Example

_if a _isnt _unset
_then
  _for x _over a.fast_elements()
  _loop
    _if x _isnt _unset
    _then
        _for child _over x.fast_elements()
        _loop
            _return child
        _endloop
    _endif
  _endloop
_else
  _return b
_endif

Compliant Solution

_if a _is _unset _then _return b _endif

_for x _over a.fast_elements()
_loop
_if x _isnt _unset
_then
    _for child _over x.fast_elements()
    _loop
        _return child
    _endloop
_endif
_endloop
